<h2>Debian</h2>
Debian es un sistema operativo (SO), el cual es una distrribución de Linux, es una de las distribuciones pioneros de Linux. Esta compuestos
de software 100% libre, esto quiere decir que sus actualizaciones, novedades, etc... Las crean una comunidad de desarrolladores voluntarios
que desean mejorar este sistema operativo, son colaboraciones a través de internet, y esto lo hace destacar de las demás distribuciones GNU.

<h3>Historia</h3>
<ul>
  <li>1993: En este año fue fubndado y anunciado el proyecto Debian por Ian Murdock, que junto a su esposa Debora le pusieron el nombre (Deb-Ian).
  Este proyecto se anuncio como un software libre, estable y seguro paraq uien lo usara.</li>
  <li>1994-1995: El proyecto crecia de a pocos, y en estos años lanzaron las primeras versiones prueba (v0.9) del sistema operativo.</li>
  <li>1996: Se lanzó la primera versión oficial de Debian, la versión Debian Buzz v1.1.</li>
  <li>1999: Se lanzaron las primeras distribuciones de Debian como lo es Corel Linux y Stormix's Linux, pero no cogieron tanta fama.</li>
  <li>2000-Actualidad: Siguieron surgiendo nuevas versiones de Debian, gracias a ser un software libre a mejorado bastante y un dato curioso,
  es que la comunidad decidió ponerle nombres de personajes de Toy Story a cada versión de Debian. Actaulmente van en la versión Trixie
  la cual seria la version Debian Trixie v13.</li>
</ul>
